Current Role and Workplace Location

I currently work as a full stack senior software developer at a private background screening company in Mount Laurel, NJ called Vertical Screen.

I help my development team along with the QA, BA, DBA, and project management teams create, update, test, and deploy various background screening applications that support our multitude of services such as drug screening, courts, MVR, employment I9 verification, licensing, education, etc. I mainly use HTML5,CSS, JS, and jQuery as my cornerstone front end languages along with Angular JS for our main client interface system Producer 2.0 and Angular Typescript for its submodule component, Client Profiles. The backend I work with is purely SQL Server and my role involves creating new schemas, tables, and procedures and test out changes in development and QA acceptance environments. We use .NET WebAPI to support our front end with back end transactions and we use some VB and Java code in rare legacy applications running under web forms (dating back to 1995 til now).
